**Description:**  
A smart home dialysis simulation engine with patient profiles, real-time vital monitoring, and risk alerts. This interactive app helps visualize dialysis flow, patient vitals, and alerts for high or low blood pressure and fluid overload.

**Features:**
- Select different **Patient Profiles** (Elderly, Cardiac, Diabetic, High Fluid Overload)
- **Real-time Alerts** based on Blood Pressure (BP) and Ultrafiltration (UF) rates
- Animated **dialysis flow simulation**
- Displays **current vitals** and risk status

**Technologies Used:**
- [Streamlit](https://streamlit.io/) for interactive UI
- [Plotly](https://plotly.com/python/) for animated visualization
- Python for backend simulation logic
